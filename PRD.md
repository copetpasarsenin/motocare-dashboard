# PRD MotoCare Dashboard

## Nama Aplikasi
MotoCare Dashboard

## Deskripsi
MotoCare Dashboard adalah aplikasi web untuk mengelola layanan servis bengkel motor. Aplikasi ini menyediakan fitur autentikasi, manajemen kategori layanan, manajemen layanan servis, dan booking servis.

## Role Pengguna

### Admin
- Login
- Mengelola kategori layanan
- Mengelola layanan servis
- Melihat semua booking
- Mengubah status booking
- Menghapus data

### User
- Register
- Login
- Melihat layanan servis
- Membuat booking
- Melihat booking miliknya sendiri
- Mengubah password

## Fitur Wajib
- Register
- Login
- Change password
- JWT Bearer Token
- Role admin/user
- CRUD kategori layanan
- CRUD layanan servis
- CRUD booking
- Validasi backend
- Validasi frontend
- Swagger UI
- Frontend dashboard
- Search dan filter
- Deploy backend dan frontend

## Fitur Bonus
- Dashboard statistik
- Chart booking berdasarkan status
- Pagination server-side
- Sorting
- Dark mode
- Export Excel
